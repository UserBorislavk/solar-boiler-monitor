# Solar Boiler GVS Monitor

**Мониторинг бойлера горячего водоснабжения с солнечным коллектором на ESP32 + ESPHome**

## О проекте

Система мониторинга температуры бойлера ГВС и солнечного коллектора.  
Устройство отслеживает:
- Температуру бойлера ГВС
- Температуру подачи и обратки солнечного коллектора
- Разницу температур ΔT на коллекторе
- Эффективность работы солнечного коллектора

## Возможности

- Точный мониторинг трёх датчиков DS18B20
- Вывод информации на OLED-дисплей 0.96"
- Интеграция с Home Assistant (автоматическая)
- OTA-обновления по Wi-Fi
- Расчёт ΔT и эффективности коллектора
- Минимальные логи (стабильная работа)

## Оборудование

- **Микроконтроллер**: ESP32 DevKit
- **Датчики**: 3 × DS18B20 (водонепроницаемые)
- **Дисплей**: OLED SSD1306 128×64 (I2C)
- **OneWire**: GPIO4
- **I2C**: GPIO21 (SDA), GPIO22 (SCL)

## Установка

1. Создай новый проект в ESPHome
2. Скопируй содержимое файла `solar-boiler-monitor.yaml`
3. Замени WiFi данные на свои (если нужно)
4. Прошей на ESP32
5. Добавь устройство в Home Assistant

|-------------------------|--------------------------------|-----------------------------|
| Бойлер ГВС             | `0xf406259244c85628`          | Температура в бойлере      |
